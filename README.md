# Hospital Management System
A console-based Java application developed using JDBC and Oracle Database to manage hospital operations such as patient management, doctor management, and appointment booking.

---

## 🚀 Features
- Add new patients
- View all patients
- View all doctors
- Book appointments
- Check doctor availability
- Oracle Database integration using JDBC
- Menu-driven console application

---

## 🛠️ Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

---
